What is SimpleElastix?

SimpleElastix is an extension of SimpleITK that offers user-friendly access to the popular image registration algorithms of the elastix C++ library. This makes state-of-the-art medical image registration really easy to do in languages like Python, Java, C# and R.

Designed For Humans

A user-friendly API facilitates rapid prototyping and reduces overhead for processing of large data sets.

Multi-lingual

Silky-smooth integration with your favorite Python, Java, R, Ruby, Octave, Lua, Tcl and C# programming tools.

Powerful

Speed up your image processing pipeline with high-performance multi-threaded C++ registration algorithms.

Open source

Developed as an open source project on Github and licensed under the permissive Apache 2.0 license.

Quickstart

Preconfigured registration methods serve as robust starting points for tuning SimpleElastix to your application.

Easy To Extend

Write your own transforms, metrics and optimizers in elastix and use SimpleElastix to call your C++ code in any of the supported languages.
