What is SimpleElastix?

SimpleElastix is an extension of SimpleITK that comes with the elastix C++ image registration library. This makes state-of-the-art medical image registration really easy to do in languages like Python, Java, C# and R.

Designed For Developers

A user-friendly API facilitates rapid prototyping and reduces overhead for processing of large data sets.

Multi-lingual

Use your favorite programming tools with native bindings for Python, Java, R, Ruby, Octave, Lua, Tcl and C#.

Powerful

Speed up your image processing pipeline with high-performance multi-threaded C++ registration algorithms.

Parameter-driven Configuration

Quickly try out different transforms, metrics and optimizers using simple, declarative parameters.

Quickstart

Preconfigured registration methods serve as robust starting points for tuning SimpleElastix to your application.

Easy To Extend

Write your own transforms, metrics and optimizers in elastix and use SimpleElastix to call your C++ code in any of the supported languages.
