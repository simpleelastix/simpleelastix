What is SimpleElastix?

SimpleElastix is a fork of SimpleITK that integrates the elastix C++ image registration library. This makes state-of-the-art medical image registration really easy to do in languages like Python, Java, Ruby and R.

Designed For Researchers

A user-friendly API facilitates rapid prototyping, reusable components and reduces overhead for large data sets.

Multilingual

SimpleElastix comes with native bindings for C++, Python, Java, R, Ruby, Octave, Lua, Tcl and C# so you can continue using your favorite tools.

Powerful

Get fast results and accelerate development with multi-threaded C++ components. Individual components can be tuned for even more speed.

Parameter-driven Configuration

A parameter map interface makes it easy to configure registration components. Try out different transforms, metrics and optimizers by means of scripting.

Quickstart

Pre-configured parameter files for rigid-, affine-, non-rigid- and non-rigid groupwise registration serve as good starting points for new users.

Easy To Extend

Write your own transforms, metrics and optimizers in elastix and use SimpleElastix to call your C++ code in any of the supported languages.

Example 1: A Line of Code

With SimpleElastix it is dead simple to align images of different patients. Elastix uses a multi-resolution strategy with Adaptive Stochastic Sampling (Klein et al. 2009) and Transform Bending Energy regularisation (Klein & Staring 2010) by default. Depending on the complexity of your problem, you can often register images with a single function call. The following is Python code.

 >>> fixedImage = SimpleITK.ReadImage("fixedImage.nii")
 >>> movingImage = SimpleITK.ReadImage("movingImage.nii")
 >>> resultImage = SimpleITK.Elastix(fixedImage, movingImage, "nonrigid")

Example 2: Robust Initialization

Difficult registration problems often need initialization to account for global translation, scaling and rotation prior to non-rigid registration. You can pass a list of configuration objects (which we refer to as "parameter maps") and have elastix run registrations sequentially using output of one registration as input for the next.

 >>> parameterMapList = SimpleITK.ParameterMapList()
 >>> parameterMapList.push_back(SimpleITK.GetDefaultParameterMap("affine"))
 >>> parameterMapList.push_back(SimpleITK.GetDefaultParameterMap("nonrigid"))
 >>> resultImage = SimpleITK.Elastix(fixedImage, movingImage, parameterMapList)

Example 3: Atlas-based Segmentation

You can apply transformations to multiple images. For example, you can transform a label image for atlas-based segmentation.

 # Non-rigid registration
 elastix = SimpleITK.SimpleElastix()
 elastix.SetParameterMap(SimpleITK.GetDefaultParameterMap())
 elastix.SetFixedImage(fixedImage)
 elastix.SetMovingImage(movingImage)
 elastix.Execute()

 # We turn off interpolation for labels
 parameterMap = elastix.GetTransformParameterMap()
 parameterMap[0]["ResampleInterpolator"] = ["FinalNearestNeighborInterpolator"]

 # Warp label image using transform from above
 resultLabel = SimpleITK.Transformix(labelImage, parameterMap)

Example 4: Configuring Registration Components

Elastix introduced the concept of a "parameter map" to specify registration components. A parameter map is a collection of key-value pairs that configure the registration procedure. Here are a couple of examples.

 parameterMap = SimpleITK.GetDefaultParameterMap("nonrigid")

 # Want to monitor registration convergence?
 parameterMap["WriteResultImageEachIteration"] = ["true"]

 # Want to use another optimizer?
 parameterMap["Optimizer"] = ["QuasiNewtonLBFGS"]

 # Want a more smooth deformation field?
 parameterMap["Metric1Weight"] = ["2.0"]

 # Want a more dense b-spline deformation field (mm)?
 parameterMap["FinalGridSpacingInPhysicalUnits"] = ["4.0"]

 # Want to make registration more robust?
 parameterMap["NumberOfResolutions"] = ["6"]
 parameterMap["NumberOfIterations"] = ["1024"]
 parameterMap["NumberOfSamples"] = ["8192"]

 # Want the raw b-spline parameters?
 bsplineParameters = elastix.GetTransformParameterMap()[0]["TransformParameters"]
