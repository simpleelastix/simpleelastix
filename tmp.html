What is SimpleElastix?

SimpleElastix is a fork of SimpleITK that integrates the elastix C++ image registration library. This makes state-of-the-art medical image registration really easy to do in languages like Python, Java, Ruby and R.

Designed For Developers

A user-friendly API facilitates rapid prototyping and reduces overhead for processing large data sets.

Multi-lingual

SimpleElastix comes with high-performance native bindings for Python, Java, R, Ruby, Octave, Lua, Tcl and C#.

Powerful

High-performance multi-threaded C++ code and tuning options for CPU and memory consumption.

Parameter-driven Configuration

Quickly try out different transforms, metrics and optimizers using simple, declarative parameters.

Quickstart

Opinionated preconfigured registration methods serve as robust starting points for many applications.

Easy To Extend

Write your own transforms, metrics and optimizers in elastix and use SimpleElastix to call your C++ code in any of the supported languages.
